.. _install:

Installation
============


Install viol
------------

First, if you have virtualenvwrapper installed (optional)::

    $ mkvirtualenv viol

then to install from the svn repository::

    $ pip install viol

*- or -* to install from a local copy of the viol package::

    $ python setup.py install


Upgrade viol
------------

To upgrade from svn::

    $ pip install -U viol
