.. _install:

Installation
============


Install invx
------------

First, if you have virtualenvwrapper installed (optional)::

    $ mkvirtualenv invx

then to install from the svn repository::

    $ pip install invx

*- or -* to install from a local copy of the invx package::

    $ python setup.py install


Upgrade invx
------------

To upgrade from svn::

    $ pip install -U invx
